# sedona-json
JSON API for Sedona VM systems

# JSON Structure

```
{
"pointId": "c.id",
"pointName": "c.name",
"pointType": "t.kit.name",
"slots": {
"slot.id": {
"name": "slot.name",
"type": "slot.type.name",
"value": "slot.isProp"
}
},
"childObjects": {
"pointId1": "point",
"pointId2": "point",
"pointI3": "point",
"pointId": "point"
}
}
{
  "slotId": {
    "1": "value",
    "2": "value",
    "3": "value",
    "4": "value"
  }
}
```
